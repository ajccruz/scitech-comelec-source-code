\subsection{Detailed Description}
A buffer for a single color plane in an uncompressed image. 

This contains the image data in a left-\/to-\/right, top-\/down format. Each row of pixels is stored contiguously in memory, but successive rows need not be. Use {\itshape stride} to compute the offset of the next row. The encoder accepts both positive {\itshape stride} values (top-\/down in memory) and negative (bottom-\/up in memory). The decoder currently always generates images with positive strides.
